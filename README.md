# go-utils
Go utils, Go语言工具，go语言中一些函数不齐全，为了方便使用，将一些自己常用的操作自己进行封装，方便使用。



# 首先下载go-utils

```go
go get -u github.com/cimuxh/go-utils
```

然后找自己需要的函数使用。

# 函数说明及使用方法

**slice**包中函数

|       函数名       |          解释          |                   用法                   |
| :----------------: | :--------------------: | :--------------------------------------: |
|  slice.MaxSlice()  |    在切片中找最大值    |        传入切片，返回最大值和错误        |
|  slice.MinSlice()  |    在切片中找最小值    |        传入切片，返回最小值和错误        |
| slice.SliceIndex() | 在切片中找某个值的位置 | 传入切片和目标值，返回下标，未找到返回-1 |
|                    |                        |                                          |
|                    |                        |                                          |
